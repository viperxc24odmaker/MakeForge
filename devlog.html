MakeForge is live

Day one. Studio name locked, brand mark designed, and the site went live on GitHub Pages — free hosting, zero cost, real URL. Everything from here builds on this foundation.

July 19, 2026
Tool Shipped

Pixel Studio v1.0

First real tool under the MakeForge name. A full sprite and animation editor built in a single file, no dependencies.

  • Pencil, eraser, fill bucket, and eyedropper tools
  • Multi-frame animation with live preview
  • Export as PNG or full spritesheet
July 19, 2026
Tool Shipped

CACIG v1.0

The Custom App Creation Idea Generator — built to solve MakeForge's own "what do I build next" problem.

  • Randomly combines type, feature, and twist into a real idea
  • Filter by category and difficulty
  • Favorites and history, saved locally
July 20, 2026
Tool Shipped

HTML Tester v1.0

A live HTML/CSS/JS playground for testing snippets before committing anything to the live site.

  • Separate HTML, CSS, and JS tabs
  • Live preview with auto-update
  • Download a standalone test file
July 20, 2026
Design

Light/dark theme switcher

Every page now supports both a dark and bright theme with a single toggle. The choice is saved automatically and stays consistent across the whole site.
